Extend command is written to facilitate the web development process.

It helps if the team members are actively using the browser and often commit. When a new extension will notify you commit, or save the data on a new commit locally.

It works with public repositories on GitHub. Private is not available (authorization to add is in the plans).

To add a repository, click the tab "Repositories", add it to the address on GitHub in the text box and click "Save". On the spot - at the same address. The next time you update the extension to obtain data on the latest commit.
